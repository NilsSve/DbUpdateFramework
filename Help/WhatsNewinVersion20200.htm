Version 2020.0.0.0 (Released Summer 2020)

  • Added new function ApiTableMoveFileListEntry
  • Added new help text to more in detail explain how the update logic works when a package is used that has been created by the DUFCodeGenerator utility.
  • All RDC Tools International's software has changed the way tools are versioned. And starring from this version it has the year as the major version number (2020 in this case). The reason behind this is that it makes it easier to have the same versioning system for all software. And it was also necessary for implementing the new installation program logic where the tools have a "Check for Updates" in the Help menu. All installation programs are now made with the Advanced Installer (same as DAW uses for DataFlex).
