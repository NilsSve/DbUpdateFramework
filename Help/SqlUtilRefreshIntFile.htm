SqlUtilRefreshIntFile - cDbUpdateFunctionLibrary

To update an .INT file when the SQL back-end table has been changed.


Type: Function


Parameters: Handle hTable

Parameter

Description

hTable

An existing filelist.cfg file number


Syntax


Function SqlUtilRefreshIntFile Integer hTable Returns Boolean

Call:

Get SqlUtilRefreshIntFile OrdHea.File_Number to bOK



Description

Use it to refresh a. .Int file when a SQL table has been changed at the SQL back end, e.g. with another tool than the Studio or The DataBase Update Framework.
