First Time Setup - The Database Update Framework

Setup


The framework uses two tools that should be added to the Studio's Tools menu before you start using it.


Note: If you used the installation program to install the framework and you are using DF 18.2 or 19 these steps should already have been done for you.


1.Install the Database Update Framework workspace in parallel with your other workspaces.

2.Obsolete! With DUF 2.0 this step is no longer needed! (Locate the Programs\CreateDbVersionTable.exe from the DbUpdateFramework workspace, start it and click the "Add to Studio" button. Adjust the 'Major Version' and 'Minor Version' spin-forms to match your Studio version. Click the 'Add Now' button to add the tool to the Studio's Tools menu. See CreateDbVersionTable for more info.)

3.Locate and start the Programs\SQLConnections.exe from the DbUpdateFramework workspace. Start it and click the "Add to Studio" button at the lower left of the program. Adjust the 'Major Version' and 'Minor Version' spin-forms to match your Studio version. Click the 'Add Now' button to add the tool to the Studio's Tools menu. See SQLConnections for more info.

4.To integrate The Database Update Framework help with the Studio's help system you need to copy the file Developer5.chm (this file) from the DbUpdateFramework workspace Help folder to the DataFlex version's Help folder;  E.g. "C:\Program Files (x86)\DataFlex 19.0\Help"
