Framework Structs

See Also: Overview of the DataFlex Database Update Framework

Framework Data Types are structured data types declared in the DataFlex framework that are not part of the DataFlex Language itself.

Please consult the Language Guide for documentation of the fundamental data types and structured types in the DataFlex language.

 

The DataFlex Database Update Framework Data Types are:

•tSQLConnection

•tSQLScriptArray

•tSqlErrorArray

•tDbUpdateError

•tColumnType

•tSQLLoggedInUser

•tAPITableNameInfo

•tAPIColumn

•tAPIIndex

•tAPIRelation
