Methods for cDbUpdateVersion

Name

Type

Description

UpdateVersionColumnValue

Procedure

Automatically called after the OnUpdate event has been executed and no errors occurred. It will update the parent container (the cDatabaseUpdateHandler) object's Data_File_Field with the value of the pnVersionNumber property. This method is considered private.


For all other methods see: cDbUpdateFunctionLibrary. All of those functions are available from a cDbUpdateVersion object.
